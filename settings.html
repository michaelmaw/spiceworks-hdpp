Ticket Counts

Ticket List Height

Adjust default height of the ticket list

Ticket View Menu

Choose which options are displayed on the ticket view dropdown menu, and in what order

Stats & Rankings

Show a toolbar above the ticket list displaying a leaderboard and ticket stats/rankings


NOTE: This feature is restricted to admin users only. Use the Pepper Roles plugin to bypass this restriction.

Default Comment Form

Select which comment form is shown by default

Delete Button

Control visibility of the ticket Delete button

Time Spent Options

Define a custom list of time spent options to override the defaults shown on the quick entry toolbar

Quick Entry Fields

Choose which fields are displayed on the quick entry toolbar, and in what order (for faster data entry)

Contact Cards

Choose what details are shown on contact cards under the ticket Details tab, and in what order

Required Fields (New Tickets)

Fields required when creating a new ticket

  • The Contact Cards and Stats & Rankings features require use of the Pepper Roles plugin for non-admins
